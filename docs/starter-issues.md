# Starter Issues

These are planned starter issues for new contributors and future maintenance work.

## Good First Issues

1. Add English quickstart examples for all skills.
2. Add fake Obsidian vault fixtures for `saodust`.
3. Add Codex-specific usage notes to each skill README.
4. Add Claude Code compatibility notes.
5. Add a fake full article example for `write`.
6. Add privacy checklist tests for examples.
7. Add a screenshot-free quickstart guide.
8. Expand validation for `skill-manifest.json`.

## Contributor Notes

Please use fake data only. Do not submit private vault paths, personal voice samples, customer details, or copied third-party workflows.
