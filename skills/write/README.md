# Write

Write is a Chinese long-form writing workflow for agent-assisted drafting.

It helps preserve:

- what the user actually said
- what external sources say
- what the AI synthesized

## Example

```text
Use write to turn this messy note into a public article draft.
```

## Best Use Cases

- WeChat articles
- product notes
- method essays
- personal operating-system notes
- public reflections with evidence

## Privacy

The public version does not depend on private voice corpora, customer records, or third-party author style instructions.
