# Four Platform Daily Content

This skill adapts one idea into Chinese drafts for:

- WeChat
- Xiaohongshu
- X
- Douyin

It treats each platform as a different communication job rather than a different word count.

## Example

```text
Turn this note into WeChat, Xiaohongshu, X, and Douyin versions.
```

## Privacy

The public version does not include private templates, customer records, personal voice files, or third-party creator style rules.
