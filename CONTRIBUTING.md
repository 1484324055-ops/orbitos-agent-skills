# Contributing

Thanks for improving OrbitOS Agent Skills.

## Good Contributions

- Fix unclear instructions in a skill.
- Add examples that use fake or public data only.
- Improve compatibility with Codex, Claude Code, OpenCode, or other agent runtimes.
- Add a new skill only when it is reusable, privacy-safe, and clearly licensed.

## Privacy Rules

Do not include:

- real customer conversations
- private vault paths
- API keys, tokens, cookies, or credentials
- screenshots containing personal data
- copied third-party prompts without permission

## Pull Request Checklist

- [ ] The skill has `SKILL.md`.
- [ ] The skill has a clear trigger and output format.
- [ ] Examples use fake or public data.
- [ ] No private paths or secrets are included.
- [ ] Third-party material, if any, is licensed and attributed.
- [ ] `npm test` passes.
