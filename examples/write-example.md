# Write Example

User request:

```text
Use write to turn this rough idea into a public article:

AI tools are getting better, but the real gap is still workflow design.
```

Expected behavior:

1. Ask clarification questions.
2. Offer outline options.
3. Search external references if current facts are needed.
4. Draft the article.
5. Keep user words, external references, and AI notes separate.

Example outline options:

- Method angle: why workflow beats tool collection.
- Case angle: how one messy task becomes reusable.
- Trend angle: AI adoption shifts from tool use to system design.
