# English Quickstart

This repository is Chinese-first, but the skills can still be reviewed and adapted by English-speaking contributors.

## Saodust

Use `saodust` when you want an agent to inspect an Obsidian vault and produce a health report.

Example:

```text
Run a dry-run knowledge-base health check. Use fake paths in the report and do not modify files.
```

## Write

Use `write` when you want a structured Chinese long-form article workflow.

Example:

```text
Use the write skill to turn this rough idea into a Chinese article outline. Keep my original words separate from AI synthesis.
```

## Four Platform Daily Content

Use `four-platform-daily-content` when you want one source adapted into multiple Chinese content platforms.

Example:

```text
Adapt this source note into WeChat, Xiaohongshu, X, and Douyin drafts. Keep platform assumptions explicit.
```
