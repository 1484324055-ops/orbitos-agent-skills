# Security Policy

## Reporting A Vulnerability

Please open a private security advisory or contact the maintainer before publicly disclosing issues that may expose private data or unsafe instructions.

## Scope

Security issues include:

- accidental secret leakage in examples
- instructions that encourage publishing private user data
- unsafe file deletion behavior
- unclear boundaries around external web sources

These skills are instruction bundles. They do not run a server and do not collect telemetry.
